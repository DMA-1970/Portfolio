# Reports

This folder contains automated reports generated by GitHub Actions workflows.

## Folder Structure

```
Reports/
├── traffic/          # Traffic monitoring reports
│   ├── YYYY/MM/      # Reports organized by year/month
│   ├── latest-summary.md
│   └── latest-report.md
├── performance/     # Performance analysis reports (Lighthouse)
│   ├── YYYY/MM/      # Reports organized by year/month
│   ├── latest-report.html
│   └── latest-summary.txt
├── links/           # Link checking reports
│   ├── YYYY/MM/      # Reports organized by year/month
│   └── latest-report.md
└── README.md         # This file
```

## Report Types

### Traffic Reports
- **Frequency**: Every 6 hours
- **Location**: `Reports/traffic/`
- **Contents**:
  - Repository views and unique visitors
  - Repository clones
  - Site uptime and response times
  - Page accessibility checks
  - Historical data organized by date

### Performance Reports
- **Frequency**: On every push and pull request
- **Location**: `Reports/performance/`
- **Contents**:
  - Lighthouse performance scores
  - Accessibility metrics
  - Best practices scores
  - SEO scores
  - Detailed HTML and JSON reports

### Link Check Reports
- **Frequency**: Daily at 6 AM UTC, and on every push
- **Location**: `Reports/links/`
- **Contents**:
  - Internal link validation
  - External link checking
  - Broken link detection
  - Link accessibility status

## Latest Reports

## Historical Reports

All reports are saved with timestamps in folders organized by year and month:
- `Reports/traffic/YYYY/MM/traffic-summary-YYYYMMDD-HHMMSS.md`
- `Reports/performance/YYYY/MM/lighthouse-report-YYYYMMDD-HHMMSS.html`
- `Reports/links/YYYY/MM/link-report-YYYYMMDD-HHMMSS.md`

## Viewing Reports

- **Markdown files** (.md): View directly on GitHub
- **HTML files** (.html): Download and open in a web browser
- **JSON files** (.json): View raw data or parse programmatically

## Automatic Updates

Reports are automatically:
- Generated by GitHub Actions workflows
- Committed to this repository
- Organized by date
- Updated with latest versions
